### Supported Languages
- C++
- C
- Python 
- Java
  
### Features
- Live autocomplete 
- 39 Themes
- Find and replace.
- Efficient workflow

### Future features
- Add your own code templates.
- Question preview from the code editor.
- Selecting your own input files. 
- Shortcuts for an efficient workflow. 
- Improved design.
